# TasteVector

TasteVector is a personalized restaurant recommendation system built as a Linear Algebra applied project.

The project uses Python, NumPy, Pandas, Flask, and pytest. Recommendation logic is organized around matrix construction, similarity, projections, SVD, eigendecomposition, and PageRank-style ranking.

## Project Structure

```text
data/
src/
app/
tests/
requirements.txt
README.md
```

## Development

Install dependencies:

```bash
pip install -r requirements.txt
```

Run tests:

```bash
pytest tests/ -v
```
